# OpenPosSuite
[![GPLv3 License](https://img.shields.io/badge/License-GPL%20v3-yellow.svg)](https://opensource.org/licenses/)

OpenPosSuite (working title) is an experimental point-of-sale system
intended for development, learning, and testing purposes.

This project is currently **NOT production-ready** and **NOT intended
for use in real commercial environments**.

While features such as fiscalization (e.g. TSE for Germany) may be added
in the future, no guarantees are made regarding legal, tax, or regulatory
compliance in any jurisdiction.

Use at your own risk.

The operator of a point-of-sale system is solely responsible for
ensuring compliance with all applicable laws and regulations.
